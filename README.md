#  AI Chatbot - Streamlit & OpenAI

Este é um projeto de chatbot interativo que utiliza a API da OpenAI para gerar respostas inteligentes em tempo real. A interface foi construída com **Streamlit**, permitindo uma experiência de usuário fluida e responsiva.

##  Funcionalidades
* **Memória de Contexto:** Utiliza `st.session_state` para manter o histórico da conversa.
* **Interface Intuitiva:** Chat inputs e mensagens formatadas nativamente pelo Streamlit.
* **Upload de Arquivos:** Suporte inicial para recebimento de arquivos via `st.file_uploader`.
* **Integração com GPT-4o:** Configurado para utilizar o modelo mais recente da OpenAI.

##  Tecnologias Utilizadas
* **Python:** Linguagem base do projeto.
* **Streamlit:** Framework para criação da interface web.
* **OpenAI API:** Engine de inteligência artificial para processamento de linguagem natural.

##  Notas de Estudo
Durante o desenvolvimento, explorei conceitos de:
- **Hugging Face:** Plataforma de modelos pré-treinados para diversas tarefas de IA.
- **LangChain:** Framework para construção de aplicações complexas com LLMs.
